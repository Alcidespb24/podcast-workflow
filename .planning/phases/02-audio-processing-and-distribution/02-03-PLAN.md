**1. Create `src/infrastructure/rss.py`:**

`build_podcast_feed(podcast_title: str, podcast_description: str, base_url: str, episodes: list[Episode]) -> str`
- Use feedgen FeedGenerator with `load_extension("podcast")`
- Set channel-level fields: title, link (base_url), description, language ("en")
- Set iTunes namespace fields: itunes_author (podcast_title), itunes_explicit ("no"), itunes_category ("Technology"), itunes_type ("episodic")
- For each episode, add entry with: id ({base_url}/episodes/{ep.filename}), title, description, published (with UTC timezone), enclosure (url={base_url}/episodes/{ep.filename}, length=str(ep.file_size), type="audio/mpeg")
- Set podcast entry extensions: itunes_duration (ep.duration_str), itunes_episode (ep.episode_number), itunes_episode_type ("full")
- Return `fg.rss_str(pretty=True).decode("utf-8")`
- Wrap in try/except, raise RSSError on failure

`validate_podcast_rss(xml_str: str) -> list[str]`
- Parse with xml.etree.ElementTree
- Check for channel element
- Check required iTunes namespace tags at channel level: {ITUNES_NS}author, {ITUNES_NS}explicit, {ITUNES_NS}category
- Check each item for: title, enclosure, guid
- Return list of error strings (empty = valid)

Create `src/infrastructure/obsidian_writer.py`:

`write_episode_to_vault(episode: Episode, mp3_source_path: str, transcript: str, vault_output_dir: str) -> tuple[str, str]`
- Build date-prefixed filename: `{episode.published_at:%Y-%m-%d} - {sanitize_filename(episode.title)}`
- MP3 filename: `{date_title}.mp3`
- Note filename: `{date_title}.md`
- Ensure vault_output_dir exists (os.makedirs(exist_ok=True))
- Copy MP3 from mp3_source_path to vault_output_dir/mp3_filename using shutil.copy2
- Build markdown note content:
  - YAML frontmatter block (use yaml.dump or manual f-string) with: title (quoted), date (YYYY-MM-DD), episode_number (int), hosts (list), style (string), source_file (string), duration (formatted string), tags (["podcast", "episode"])
  - H1 heading with episode title
  - Obsidian wiki link: `![[{mp3_filename}]]`
  - Blank line
  - Foldable transcript callout: `> [!note]- Transcript` followed by each transcript line prefixed with `> `
- Write note to vault_output_dir/note_filename (UTF-8)
- Return (mp3_dest_path, note_dest_path) tuple
